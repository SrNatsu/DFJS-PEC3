# Ejercicio 1

## Dentro de la carpeta ```PEC3_Ej1``` crea un fichero Markdown que tenga como nombre ```PEC3_Ej1_respuestas_teoria.md``` en este fichero enumera y explica las principales ventajas que aporta el uso de TypeScript sobre JavaScript.

1. **Tipado estático y seguro:** Permite definir tipos de datos para variables, funciones y objetos, lo que evita errores comunes como pasar un tipo de dato incorrecto.
2. **Detección temprana de errores:** Al compilar el código, TypeScript identifica errores de sintaxis y tipado antes de que el código se ejecute, a diferencia de JavaScript que los detecta al ejecutarlo.
3. **Mejor herramienta y autocompletado:** Los editores como VS Code ofrecen autocompletado avanzado, navegación de código y refactorización más segura gracias al tipado estático.
4. **Escalabilidad y mantenimiento:** Es ideal para proyectos grandes y equipos, ya que el código es más estructurado, auto-documentado y fácil de entender.
5. **Características avanzadas:** Incluye funcionalidades como interfaces, enumeraciones, gnéricos y clases con modificadores de acceso, facilitando la orientación a objetos.
6. **Superconjunto de JavaScript:** Todo código JavaScript válido funciona en TypeScript, permitiendo una migración gradual y uso de librerías JS existentes.
